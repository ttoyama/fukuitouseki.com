## 利用方法

これらの資料は福井県の災害時透析マニュアル作成時の参考資料として活用されます。各都道府県の特色やアプローチの違いを比較検討し、福井県の実情に合った内容を策定するために使用します。

## 注意事項

- 著作権に配慮し、必要に応じて出典を明記
- 公開可能な資料のみを収集
- 定期的な資料の更新状況確認を推奨
